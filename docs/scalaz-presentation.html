Структуры данных
  • Validation Represent computations that may success or fail, accumulating multiple errors.
  • NonEmptyList A list containing at least one element.
  • DList A difference list, supporting efficient append and prepend.
  • EphemeralStream A stream that holds weak references to its elements, and recomputes them if needed if reclaimed by the garbage collector.
  • Heap A priority queue, implemented with bootstrapped skew binomial heaps.
  • Endo Represents functions from A => A.
  • FingerTree A tree containing elements at it's leaves, and measures at the nodes. Can be adapted to various purposes by choosing a different measure, for example IndSeq and OrdSeq.
  • Lens Composable, functional alternative to getters and setters
  • Tree A multiway tree. Each node contains a single element, and a Stream of sub-trees.
  • TreeLoc A cursor over a Tree.
  • Zipper A functional cursor over a List.
  • Kleisli Represents a function A => M[B], allowing chaining. Also known, and aliased, as scalaz.ReaderT.
  • Transformers: StateT, WriterT, OptionT, EitherT

Функторы → Монады
trait Functor[F[_]] {
  def fmap[A, B](f: A => B): F[A] => F[B]
}
trait Pointed[P[_]] extends Functor[P] {
  def pure[A](a:A):P[A]
}
trait Applicative[T[_]] extends Pointed[T]{
  def <*>[A,B](tf:T[A=>B])(ta:T[A]):T[B]
}
trait Monad[M[_]] extends Applicative[M]{
  def >>=[A,B](ma:M[A])(f:A=>M[B]):M[B]
}
